SETTING UP THE SYSTEM
---------------------
```
> SERVER_2/npm install
> SERVER_3/npm install
> SERVER_1/npm install
```

HOW TO RUN THE APP
------------------
START THE SERVER IN THE BELOW ORDER:
  - SERVER 2
  - SERVER 3
  - SERVER 1

TO START THE APP
----------------
```
> SERVER_2/npm start
> SERVER_3/npm start
> SERVER_1/npm start
```
